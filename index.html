Radar Chart

Graduate Project Tutorial by Neha Rao

    Overview

    Radar charts are used to visualize multi-variate that would otherwise appear either overwhelming or underwelming using conventional graphing methods (static charts)

    For this lab, we will be creating a radar chart that compares the average consumer expenditures between two regions. (The Midwest and another).

    Since this uses a d3.js file, you will be incorporating the data into the HTML code itself. Click on d3.js to learn more data driven documents.

    Please Note: unlike the other labs, you will not be able to preview unless you've finished adding all the lines of code. But since the code modification is minimal it should not be an issue!

    5. Customize and Render Chart

    Now, all we have left is to customize the chart and call out the function that actually draws it!

  5. Scroll to the DRAW THE CHART section and add the following lines. 
    
                var color = d3.scale.ordinal()
				    .range(["#CC333F","#00A0B0"]);
				
			     var radarChartOptions = {
                  w: width,
                  h: height,
                  margin: margin,
                  maxValue: add value, //maximum value (temp value)
                  levels: 5, //number of breaks
                  roundStrokes: true,
                  color: color
			};

});
  6. The var color = d3.scale.ordinal() attributes colors in the same order as the data presented. So the first color code (#CC333F) will be attributed to the data 1, since it's the first in the code. Just something to know! :)
  7. maxValue will be the highest values in you data. For now, it is anything above 22,500, since the highest expediture value we have 22,612
  8. I've left the level to 5, which are the breaks int the data but of course, you can change them if it's too many or too less

  9. Finally, scroll all the way to the bottom and add the following lines below the CALL FUNCTION FOR RADAR CHART section: 
    
                    RadarChart(".radarChart", data,radarChartOptions);
});
  10. If you study the radarChart.js, you'll see that this function is defined at the very beginning, which is how we're able to call it out.

  11. Now, you should be able to preview it!
